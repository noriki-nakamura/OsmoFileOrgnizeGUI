# OSMO Pocket データ整理ツール

OSMO Pocket 等のカメラから SD カード内のメディアファイルを読み込み、撮影日やファイル形式に基づいて自動的にフォルダ分け・バックアップを行うデスクトップアプリケーションです。

## 主な機能

- **高度な自動分類**:
    - **撮影日による整理**: 動画・写真のメタデータを解析し、撮影日ごとにフォルダを自動作成（日付形式は `YYYYMMDD` と `YYYY-MM-DD` から選択可能）。
    - **種類による整理**: 動画、写真、それ以外のファイルを任意のフォルダ名（例: Videos, Photos, Misc）に分類。
    - **拡張子による整理**: ファイル拡張子（MP4, JPG, DNG等）ごとのサブフォルダ作成。
- **柔軟なカスタマイズ**:
    - **階層順序の変更**: 「日付 > 種類」や「種類 > 日付 > 拡張子」など、フォルダの階層構造を自由に選択可能。
    - **対象拡張子の指定**: どの拡張子を動画・写真として扱うかをユーザーが自由に変更。
    - **除外設定**: `.LRF` や `.THM` などのプロキシ・サムネファイルを指定してコピー対象から除外。
- **便利な機能**:
    - **リアルタイム表示**: 設定変更が即座にプレビューリストに反映。
    - **コピー完了後の自動処理**: SDカードからの自動消去、および保存先フォルダの自動オープン（エクスプローラー）。
    - **進行状況の可視化**: プログレスバーと進捗ラベルによるコピー状況の確認。

## セットアップ手順

### 前提条件
- Python 3.8 以上がインストールされていること。

### インストール
リポジトリをクローンまたはダウンロードし、プロジェクトディレクトリで以下を実行して依存パッケージをインストールします。

```bash
uv sync
```

## 使い方

1. `main.py` を実行してアプリケーションを起動します。
   ```bash
   uv run main.py
   ```
2. **「読み込み元」**: SDカードまたはソースフォルダを選択します。
3. **「保存先」**: 整理後のファイルをコピーするディレクトリを選択します。
4. **「整理設定」**: 日付形式やフォルダ名、階層の順位などを好みに合わせて調整します。
5. 右側の **「整理後のファイル構成（プレビュー）」** を確認し、問題なければ **「整理・バックアップを開始」** をクリックします。

## 技術スタック
- **GUI**: Python 標準ライブラリ `tkinter`
- **メタデータ取得**: `mutagen` (MP4), `Pillow` (JPEG/EXIF)
- **ファイル操作**: `shutil`, `pathlib`
- **テスト**: `pytest`, `pytest-cov`

## 開発者向け: ユニットテスト
